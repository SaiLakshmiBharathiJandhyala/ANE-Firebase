Property Detail
invitationIdsproperty
public var invitationIds:Array

This array will contain a series of String values each being the identifier of an invite.

When an invite is sent this may contain several identifiers for each person the invite was sent to.

When an invite is received this should only contain one identifier being the invite that was used to open or install the app.
